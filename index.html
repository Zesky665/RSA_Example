Instructions

#1 : Enter student ID into the Student ID text box.

#2 : Press Generate Keys, this will generate a public and private key based on the input.

#3 : Press Encrypt to encrypt the Student ID with the public key.

#4 : Copy the number in output and paste it into the Student ID text input, then press Decrypt to decrypt the Student ID with the private key.


Misc

#1 : The two primes used in this app are 19 and 29

#2 : The algorithm assumes that the input is 4 digits long and numerical when using the encrypt function.

#3 : The algorithm assumes that the input it 6 digits long and numerical when using the decrypt function.
